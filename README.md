# smp-pelita-bangsa-website
# Web Informasi SMP Pelita Bangsa Pamulang

Website informasi sekolah yang dirancang untuk memberikan informasi lengkap mengenai SMP Pelita Bangsa Pamulang kepada siswa, orang tua, dan masyarakat.

## Fitur

- Beranda
- Profil Sekolah
- Visi dan Misi
- Informasi Guru
- Informasi Siswa
- Galeri Kegiatan
- Berita dan Pengumuman
- Kontak Sekolah

## Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript
- Bootstrap
- PHP
- MySQL

## Tujuan

Membangun sistem informasi sekolah yang dapat mempermudah penyampaian informasi akademik dan non-akademik secara digital.
